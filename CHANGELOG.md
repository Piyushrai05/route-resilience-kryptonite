# Changelog

All notable changes to the **Route Resilience** project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-01

### Added
- **Unified Layout**: Developed Navbar, Sidebar, and Status Bar components.
- **Leaflet GIS Maps**: Dynamic satellite and street layer toggle, measure tools, and custom markers.
- **AI Road Extraction Slider**: Side-by-side comparison sliding viewport with ML evaluation metrics.
- **Graph Healing Pipeline**: Algorithmic DSU and MST connectivity healing controllers.
- **Disaster Simulation**: Dynamic flood circles, delays charts, and alternative routing rendering.
- **Reports Export Suite**: PDF print brief configurations and CSV/GeoJSON downloads.
- **Mock APIs**: Standardised endpoint responses for uploads, segmentation, healing, and simulation.
